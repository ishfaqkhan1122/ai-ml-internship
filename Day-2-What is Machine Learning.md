# What is Machine Learning?

## Definition (Simple English)

**Machine Learning (ML)** is a branch of **Artificial Intelligence (AI)** that enables computers to **learn from data** and **improve their performance without being explicitly programmed for every task**.

---

## Easy Example

Imagine you want a computer to recognize **cats in photos**.

### Traditional Programming

You write rules such as:

- If it has pointed ears
- If it has whiskers
- If it has a tail

Then the computer decides:

> **"It's a cat."**

### Machine Learning

Instead of writing rules manually:

- You show the computer **thousands of cat and non-cat images**.
- The computer **learns the patterns by itself**.
- It can then **identify new cat images** that it has never seen before.
